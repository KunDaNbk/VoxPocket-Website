# VoxPocket 宣传页面 PRD 文档

## 1. 产品概述

VoxPocket 是一款基于 llama.cpp 的本地 AI 聊天 Android 应用，让用户能够在手机上离线运行大语言模型，无需网络连接，保护隐私。

**核心价值**：
- 完全本地运行，无需网络
- 支持 GGUF 格式模型
- 隐私安全，数据不离开设备
- 优雅的深色主题 UI
- Markdown 格式渲染
- 支持思考过程显示

**目标用户**：
- 注重隐私的用户
- 开发者和技术爱好者
- 需要离线 AI 助手的用户

## 4. 用户界面设计

### 4.1 设计风格

**配色方案**：
- 主色：#7C4DFF（VoxPocket 紫色）
- 次要色：#6200EA（深紫色）
- 背景：#121212（深色主题）
- 文字：#FFFFFF（白色）
- 次要文字：#B0B0B0（灰色）
- 强调色：#03DAC6（青色）

**设计风格**：
- 深色科技风格
- 渐变背景
- 卡片式布局
- 圆角设计
- 微妙阴影效果

**字体选择**：
- 标题：现代无衬线字体（如 Poppins）
- 正文：清晰易读字体（如 Inter）

**动效设计**：
- 页面滚动动画
- 卡片悬停效果
- 按钮交互效果
- 渐入渐出动画

### 4.2 响应式设计

- 桌面端优先设计
- 平板适配
- 移动端友好
- 触摸优化

### 4.3 页面布局

| 页面区域 | 模块名称 | UI 元素 |
|---------|---------|--------|
| Hero | 主视觉区域 | Logo、标题、描述、CTA按钮、渐变背景 |
| Features | 功能卡片 | 6个特性卡片、图标、描述文字 |

## 5. 技术实现要求
